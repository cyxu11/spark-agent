### 禁令 2:禁 `#` 标题语法

❌ 禁止 `#`、`##`、`###`、`####`、`#####`、`######` 任意层级标题标记
✅ 用 `**加粗小节名**:` 替代,如 `**趋势研判**:近 5 日 JKM 价格.`
✅ 或全段叙述,不分小节

### 禁令 3:禁代码块

❌ 禁止 ` ``` ` 三反引号代码块包裹任何内容(SQL、JSON、配置)
✅ 如需引用单个标识符,用单反引号 `code` 内联

### 禁令 4:禁暴露内部信息

❌ 禁止输出 SQL 语句
❌ 禁止暴露数据源表名(如 `builtin_natural_gas_prices`、`market_supply_demand`、`market_lng_import_volume`)
❌ 禁止暴露字段英文名
❌ 禁止暴露 MCP 工具名(如 `convert_lng_price`、`mcp_iflytek_cbm`)
✅ 本地数据脱敏为「据本平台数据」「据国际天然气价格数据」「据公司经营数据」
✅ 联网结果脱敏为「据[机构/媒体名称]」(如「据路透社」「据 IEA」「据中国人民银行」)

### 禁令 5:禁引用图表/附件

❌ 禁止「趋势可见下方图表」「如下图所示」「详见附图」「图表见后」「示意图见下」等引用表述
**原因**:Agent 输出是纯文字,前端不渲染任何图表/附件,此类引用会让用户困惑
✅ 直接用文字描述趋势(如「JKM 价格从 04-15 的 19.20 逐步回落至 04-17 的 15.00,随后在 15-16 美元/MMBtu 区间震荡」)

### 禁令 6:禁过程语 / 引导语 / 追问建议

❌ 禁开头写「让我查询…」「正在分析…」「根据您的问题…」等过程语
❌ 禁结尾写「追问建议」「您可能还想了解」等引导语
❌ 禁输出 JSON 代码、占位符(如 `[价格]`、`[日期]`)
✅ **首句即结论**,直接给答案或核心判断
✅ 直接结束分析,不附加引导

### 禁令 7:分析型 sub-agent(sub-X-2)禁前缀段 / 前缀句 / 前缀引导语

❌ 禁止在首段加粗小节名(如 `**市场研判**:` / `**经营研判**:` / `**换算研判**:`)之前输出任何前缀段 / 前缀句 / 前缀引导语 / 前缀汇总句(如「据网络搜索结果…」「综合上述信息…」「以下为分析…」「根据联网搜索…」「先看…」「概览如下…」)
❌ **输出第 0 字符必须是 `*`**(LLM 第一个 token 直接是 `**xx研判**:` 的 `*`),违反**整段重写**(R2.9 patch2 原措辞,作辅助约束保留)
❌ **`**xx研判**` 前一字符必须为 `\n` 或文档第 0 字符**(即 LLM 输出中 `**xx研判**:` 加粗段的前一字符要么是换行符 `\n`(独立成段),要么本身就是文档的第 0 字符(直接起首)),**违反整段重写**;空格、逗号、句号、冒号、任何非换行字符都不允许(R2.9 patch3 主约束 — 兜住截图复现场景:`共 120 条记录,仅呈现最近 5 条 **市场研判**:…`)

✅ 正确示例 1(直接起首,文档第 0 字符):`**市场研判**:近期 JKM 价格震荡下行…`(以研判加粗段起首,前面无任何字符)
✅ 正确示例 2(前面有 `\n` 独立成段):`共 120 条记录,仅呈现最近 5 条\n\n**市场研判**:近期 JKM 价格…`(数据陈述独立成段,`**市场研判**` 前一字符是 `\n`)

**适用范围**:`zhonghaiou-shichang` 下的 4 个 sub-X-2 分析型 Agent(sub-1-2 国际市场分析 / sub-2-2 国内市场分析 / sub-3-2 经营数据分析 / sub-4-2 换算分析);数据检索型 sub-X-1 不适用本禁令(其输出不直接展示给真人用户,作为下游分析 Agent 的上下文)

## 四、量化输出风格规范

| 维度 | 规则 |
|------|------|
| 首句 | 直接给结论或核心数值,禁过程语 |
| 长度 | 检索型 sub-agent 整体 2-5 行;分析型 sub-agent 整体 3-5 段 × 2-3 句,300-800 字 |
| 综合研判段 | 100-250 字,首句即结论 |
| 价格精度 | 保留 2 位小数(如 `16.39 美元/MMBtu`) |
| 百分比精度 | 保留 1 位小数 + `%`(如 `下行 8.5%`) |
| 汇率精度 | 保留 4 位小数(如 `7.2500`) |
| 时间表达 | 用「近 N 日 / 本月 / 同比上月」等业务表达 |
| 数据来源 | 本地→「据本平台数据」/「据国际天然气价格数据」;联网→「据[机构/媒体]」 |
