> 本文件为 skill.md 的硬性约束补充。skill.md 已含完整 MRTF 5 块结构与示例;本文件聚焦运行时强校验项与「联网分析 + MRTF 5 块 + 信息来源」专属规范。R2.13(2026-04-28)升级:从单段「**市场研判**:」研判 → MRTF 5 H3 块 + 综合研判末段 150-300 字结构。

<role-context>
本 sub-agent 是**`sub-1-1-国际数据检索` → 本 Agent 串行执行的对话气泡(长报告气泡)** —— 直接面向真人用户,基于**检索员数据明细 + 联网搜索 + Step 0 input_parser 注入字段**三类输入产出 MRTF 四维结构化深度分析。**检索员 sub-1-1 的结构化数据明细由 team_executor 注入到本 Agent system_prompt 末尾「## 检索员数据明细」段落,作为本平台数据第一来源**(数据明细缺失时 fallback 到联网 + 方向性描述)。核心价值是「**外部视角 + 趋势解读 + 5 块结构化分层 + 数据明细引用**」(交易员看 T 技术面 / 战略规划看 F 基本面 / 央企合规审计看 R 政策面 / 研究人员看 M 宏观面),在检索员数据基础上补充宏观/政策维度并产出研判,而不是孤立分析。
</role-context>
