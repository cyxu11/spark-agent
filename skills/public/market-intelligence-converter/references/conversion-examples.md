# 换算 skill 三种问题类型正反例

## 类型 1 示例:计算性问题(`mcp_price_converter` 单工具)

### 示例 1-A:JKM 实时价换算

**用户 query**:当前 JKM 价格换算成国内出站价是多少

**工具调用**:`mcp_price_converter(category="JKM")`
**工具返回(示意)**:
```
{
  "params": {"exchange_rate": 7.2356, "ton_to_mmbtu": 52.08, "vat": 0.09, "terminal_fee": 311.3, "rebate_base": 1539.4, "rebate_ratio": 0.80},
  "result": {
    "input_price": 12.586,
    "input_unit": "USD/MMBtu",
    "data_date": "2026-04-14",
    "outputs": {
      "cost_no_tax_yuan_per_ton": 4742,
      "ex_terminal_with_tax_yuan_per_ton": 5480,
      "rebate_yuan_per_ton": 230,
      "ex_terminal_after_rebate_yuan_per_ton": 5250
    }
  }
}
```

**输出**:

按 JKM 12.59 美元/MMBtu(汇率 7.2356,2026-04-14,据本平台换算工具)计算,LNG 出站价含税约 5480 元/吨,退税后约 5250 元/吨。

| 项目 | 数值 | 单位 |
|------|------|------|
| 输入价格 | 12.59 | 美元/MMBtu |
| 汇率 | 7.2356 | CNY/USD |
| 成本不含税 | 4742 | 元/吨 |
| 出站价含税 | 5480 | 元/吨 |
| 退税额 | 230 | 元/吨 |
| 出站价退税后 | 5250 | 元/吨 |

公式说明:成本不含税 = JKM × 52.08 × 汇率;出站价含税 = 成本不含税 × (1 + 9%) + 311.3 元/吨(接收站费);退税额 = max(0, (成本不含税 − 1539.4) × 9% × 80%)。

业务提示:当前 JKM 价位下退税额 230 元/吨,体现十五五退税新政策红利;建议采购部门关注 JKM 12.5-12.8 美元/MMBtu 作为补单参考区间。

## 类型 2 示例:概念性问题(`mcp_iflytek_cbm` 单工具)

### 示例 2-A:什么是热值

**用户 query**:什么是 LNG 的热值,如何计算

**判断**:命中 "什么是" + "如何计算" 两个类型 2 关键词 → **必走类型 2,严禁调 `mcp_price_converter`**

**工具调用**:`mcp_iflytek_cbm(query="LNG 热值 定义 国际惯例")`
**输出**:

LNG 热值是衡量天然气燃料能量密度的核心指标,通常用 MMBtu(百万英热单位)或 GJ(吉焦耳)表示。1 吨 LNG 标准热值约 52.08 MMBtu,等价 54.86 GJ;具体数值因 LNG 来源、组分(甲烷/乙烷/丙烷比例)而略有差异。据 [国际能源署 IEA](https://www.iea.org/...),LNG 热值是国际贸易计价的统一基准 — 美元/MMBtu 是 JKM/TTF/HH 等基准价的通用单位。

关键要点:
- **1 吨 LNG ≈ 52.08 MMBtu ≈ 54.86 GJ**(标准热值假设)
- **甲烷为主**(>95%):热值偏高;高乙烷/丙烷组分:热值略高
- **国际贸易**:统一用 美元/MMBtu 计价,跨市场比价
- **国内换算**:1415 m³/吨(液态密度)用于到岸价 → 元/吨 转换

业务关联:对集团进口长协采购的意义在于,准确的热值参数确保跨市场比价(JKM vs TTF vs HH)和国内挂牌价换算(元/吨)在统一能量单位下进行,避免成本核算偏差。

## 类型 3 示例:综合性问题(两个工具都调)

### 示例 3-A:JKM 概念 + 当前换算

**用户 query**:什么是 JKM 价格,当前换算成元/吨多少

**判断**:既命中 "什么是" + 又含 "当前换算" → 类型 3

**工具调用**:
- `mcp_iflytek_cbm(query="JKM 价格 定义 国际惯例")`
- `mcp_price_converter(category="JKM")`

**输出**:

JKM(Japan Korea Marker)是亚洲 LNG 现货基准价,反映东北亚 LNG 现货市场供需。据 [Platts/S&P Global Commodity Insights](https://www.spglobal.com/...),JKM 由 S&P Global 每日发布,以日韩离岸价为基础。当前 JKM 报 12.59 美元/MMBtu(2026-04-14,据本平台换算工具),按汇率 7.2356 换算,出站价含税约 5480 元/吨,退税后约 5250 元/吨。

| 项目 | 数值 | 单位 |
|------|------|------|
| JKM 当前价 | 12.59 | 美元/MMBtu |
| 汇率 | 7.2356 | CNY/USD |
| 出站价含税 | 5480 | 元/吨 |
| 出站价退税后 | 5250 | 元/吨 |

JKM 的关键意义:① 亚洲 LNG 现货定价基准;② 集团长协与现货比价的核心参考;③ 与 TTF(欧洲)、HH(美国)形成跨市场比价体系。建议采购部门将 JKM 12.5-12.8 美元/MMBtu 视为当前补单参考价位,关注退税新政策下现货采购的相对优势。
