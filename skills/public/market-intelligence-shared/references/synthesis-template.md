# 末位 U 形提示(撰写最终答案的硬约束)

> 本文件是工具检索完成后,撰写面向用户的最终答案前必须遵守的硬约束。所有市场态势业务 skill 共享。

## 一、Must-Do 硬约束

1. **首句即结论** — 第一句直接给核心判断或答案(禁过程语「让我查询」「正在分析」)
2. **文字陈述优先** — 用自然语言陈述数据,数据示例:`2026 年 4 月 JKM 均价 12.50 美元/MMBtu,环比上涨 3.2%`
3. **精炼** — 数据段 2-5 行;MRTF 分析段整篇 3-5 段 / 800-1500 字
4. **数据精度**:
   - 价格/金额保留 2 位小数(`16.39 美元/MMBtu`)
   - 国内挂牌价整数(`5800 元/吨`)
   - 百分比带 `%` 保留 1 位小数(`下行 8.5%`)
   - 汇率保留 4 位小数(`7.2500`)
5. **来源引用**:
   - 联网用 markdown link `据 [机构/媒体名称](URL)`(URL 必须来自工具实际返回的 candidate_pool)
   - 本地用「据本平台数据」/「据 XX 数据」
6. **🔴 数据引用硬约束(R2.14-Patch1 — 违反 = 整段重写)**:
   - 答案**必须包含 ≥ 3 处具体数值**(日期 / 价格 / 百分比 / 万吨 / 亿立方米 / 元/吨 等)
   - 每个数值**必须来自工具检索结果**
   - **禁** 出现「相关数据」「具体数据」「详细数据」「相应信息」等空话词代替具体数值
   - 引用本平台数据的数值末尾**必须追加 `[数据]` 标识**
   - 触发自检:数着自己输出里的具体数值 — **< 3 个 → 重写**
7. **🔴 完整性硬约束(R2.14-Patch1)**:
   - 答案必须**完整覆盖用户问题的所有要点**
   - 对比类问题必须给出对比的所有对象数值;趋势类必须给出起止数值 + 涨跌幅;占比类必须给出至少 3 个分项数值
   - **禁** 仅给方向性结论(如「价格波动较大」「整体平稳」「呈上涨趋势」)而不带数值

## 二、Must-Not-Do 硬约束

1. ❌ Markdown 表格 / `|` 字符 / `<table>` HTML / 伪表格(`日期: 值` 枚举) — 仅 retriever 数据段 + MRTF 的 T/F 章节有限豁免
2. ❌ 暴露 SQL / 数据源表名 / 字段英文名 / MCP 工具名 / 内部组件名
3. ❌ `#` 标题语法 / ` ``` ` 代码块 — MRTF 5 H3 块豁免
4. ❌ 引用图表/附件(「如下图」「详见附图」「趋势可见下方图表」)
5. ❌ 过程语(「让我查询」「正在分析」「根据您的问题」)
6. ❌ 引导语(「追问建议」「您可能还想了解」)
7. ❌ 编造数据 / 占位符(`[价格]`、`[日期]`)
8. ❌ MRTF 分析段:`**xx研判**` 之前的前缀段/前缀句(禁令 7)

## 三、正反示例

### ✅ 正确输出 1(JKM 价格走势)

当前 JKM 价格为 12.50 美元/MMBtu(2026-04-24)[数据]。近 5 日从 04-20 的 12.80 高点回落 2.3% [数据],主要受亚太需求转弱和美国 LNG 出口增加影响。据 [ICIS](https://example.com/icis) 报道,欧洲补库存节奏放缓亦对价格形成压制。TTF 同期下跌 1.8%,至 34.20 欧元/MWh [数据]。

(✅ 含 5 处具体数值,> 3 处约束达标)

### ✅ 正确输出 2(LNG 挂牌价对比)

华东 LNG 挂牌价 5,420 元/吨(2026-04-30)[数据],华南 5,580 元/吨 [数据],价差 160 元/吨,华南高 3.0% [数据]。近 7 日华东累计上涨 1.8%,华南上涨 2.3% [数据],均处于近半年偏低位。据本平台数据,5 月 6 日两地价差扩大至 200 元/吨。

(✅ 含 6 处具体数值)

### ❌ 反例 1(完整性失败)

JKM 价格近期波动较大,主要受亚太需求和美国 LNG 出口影响,需关注后续走势。

(❌ 0 处具体数值,违反硬约束 → 整段重写)

### ❌ 反例 2(空话替代数据)

近一周 LNG 挂牌价整体平稳,各区域价格存在差异,详细数据见相关报告。

(❌ 0 处具体数值 + 含「相关报告」空话 → 整段重写)

### ❌ 反例 3(只方向不数值)

中海油 2024 年 LNG 进口同比有所增长,长协占主导地位,现货比例略有上升。

(❌ 仅方向性,无具体万吨数 / 比例数 → 整段重写)
