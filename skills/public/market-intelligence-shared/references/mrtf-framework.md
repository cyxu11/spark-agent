# MRTF 四维分析框架(详细模板)

> 本文件是 MRTF 分析段(`analyst` 风格)的强制输出模板。供 `market-intelligence-international` / `market-intelligence-domestic` / `market-intelligence-company` 三个业务 skill 引用。`market-intelligence-converter` 是确定性换算,不走 MRTF。

## 一、MRTF 四维总览

| 维度 | 章节标题 | 内容范围 | 主要数据来源 |
|------|---------|---------|------------|
| **M-Macro 宏观面** | `### 一、宏观面 (M-Macro)` | GDP / PPI / CPI / 汇率 / 利率 / 地缘 / 季节性 | **联网搜索为主** |
| **R-Regulation 政策面** | `### 二、政策面 (R-Regulation)` | 监管政策 / 关税 / 配额 / 补贴 / 双碳 / OPEC+ / 各国能源局 / 发改委文件 | **联网搜索为主** |
| **T-Trend 技术面** | `### 三、技术面 (T-Trend)` | 价格走势方向 / 波动区间 / 支撑压力位 / 价差结构 / 季节性 | **本平台数据为主** |
| **F-Fundamental 基本面** | `### 四、基本面 (F-Fundamental)` | 供需平衡 / 库存 / 产量 / 开工率 / 贸易流 / 消费结构 / 终端用户 | **本平台数据 + 联网补充** |
| **末段综合研判** | `**综合研判**:`(加粗节名,**不**用 `### 五、`) | 整体方向性结论 + 风险机遇 + 操作建议 | 上四维综合 |

## 三、五大强制约束(S0 - S4)

### S0 — 数值/日期幻觉零容忍(优先级最高,违反整篇作废)

- **禁止输出本平台数据明细段落不存在的任何具体日期、年份、价格、数量**
- **禁止把「近 7 日 / 近一周 / 近一月 / 最新」转写成不在数据明细里的具体日期序列**;若数据只给到 `2026-04-14`,**不准**在 T 章节表格里出现 `2026-04-15` 及之后日期
- **强制原样复述(T 章节第一表格)**:T 技术面的第一个 Markdown 表格,必须严格复制本平台数据明细段落的前 N 行(N ≤ 10),**列名 / 日期 / 数值原样照搬,禁改写、禁补全、禁推算**
- 本平台数据缺失时 → 表格用「本段无可引用本平台数据」占位,禁编造
- 业务化时间表达豁免:仅「近 7 日 / 本月 / 同比上月 / 同期 2025 年」等模糊概念允许出现在分析正文,**不允许伴随具体未引数据的日期/数值**

### S1 — 5 个 H3 章节标题中文数字编号

✅ `### 一、宏观面 (M-Macro)` / `### 二、政策面 (R-Regulation)` / `### 三、技术面 (T-Trend)` / `### 四、基本面 (F-Fundamental)`
❌ 禁阿拉伯数字 `### 1. 宏观面` / 禁罗马数字 `### Ⅰ、宏观面` / 禁纯英文 `### M. Macro`

### S2 — 综合研判末段 150-300 字 ±20%(三要素必齐)

三要素:**方向性结论 + 风险机遇 + 操作建议**
- 末段以 `**综合研判**:` 加粗节名起首
- `**综合研判**` 前一字符必为 `\n` 或文档第 0 字符

### S3 — Markdown 表格数据行 ≥ 3 行(不含表头)

- 双边对比场景允许 2 行
- 1 行迷你表格 → 整段重写
- 至少在 T / F 章节使用 1 个 Markdown 表格

### S4 — 联网关闭场景字数下限

联网开关关闭(`candidate_pool` 为空)时:
- M / R 章节标注「本段联网资讯未启用,仅基于本平台数据」**且** 章节正文 ≥ 50 字(避免空 H3)
- T + F 章节合计字数 ≥ 400 字
- 综合研判末段长度沿用 150-300 字

## 四、综合研判末段三要素细化

### 要素 1:方向性结论(首句)

✅ `近期国内 LNG 挂牌价整体延续震荡偏弱格局,核心驱动是 JKM 国际现货走弱传导 + 采暖季尾声需求边际回落…`
❌ `近期国内 LNG 价格波动较大,需关注后续走势。`(方向性模糊 + 无具体驱动)

### 要素 2:风险机遇(各 1-2 条)

✅ `**风险**:① 极端天气可能短期推升居民/发电用气;② 国际地缘冲突若升级通过 JKM 传导推高进口成本。**机遇**:① 工业「煤改气」推进利好长期需求增长;② 现货低位有利于补单。`
❌ `存在一定风险和机遇。`(空话)

### 要素 3:操作建议(按用户画像)

- 交易员/采购:**价格区间 / 补单窗口 / 套利机会**
- 战略规划:**长期需求趋势 / 区域布局**
- 央企合规审计:**政策落地节奏 / 配气价改革 / 长协占比目标**
- 销售部门:**区域价差 / 客户结构调整**
