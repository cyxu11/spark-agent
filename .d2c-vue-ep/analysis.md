## 8. 实现说明（给 codegen）

**目标文件**（非 D2C 新文件，修改已有文件）：
1. `frontend/src/core/i18n/locales/zh-CN.ts` — 改 flashMode/proMode，加 useSkillBtn
2. `frontend/src/core/i18n/locales/en-US.ts` — 同步英文
3. `frontend/src/components/workspace/input-box.tsx` — 改按钮样式 + 加使用技能

**按钮样式规格**：
- 高度：h-8（32px）
- 圆角：rounded-[6px]（约 9.6px 对应 Tailwind 6px，接近 DSL）
- 边框：border border-[#E8ECF2]
- 默认背景：bg-white
- 文字：text-sm font-normal text-[#161C23]
- 图标：16×16，用 DSL PATH 数据
- 激活（ultra/研报生成）：bg-[#EDF2FF] border-[#BAD5FE] text-[#3284FF]
- 其他激活：bg-blue-50 border-blue-300 text-blue-600（保持现有浅蓝逻辑）
- 间距：gap-1.5（icon 与文字）

**使用技能按钮**：
- 使用 DropdownMenu 包裹
- 调用 useSkills() 取 enabled=true 的技能列表
- 点击技能 → router.push('/workspace/chats/new?mode=skill&skill=name')
- 无技能时显示"暂无可用技能"
